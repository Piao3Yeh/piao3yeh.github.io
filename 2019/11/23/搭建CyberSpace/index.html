搭建博客

介绍

该博客是使用GitHub所带的gitpage来做的博客,博客上所有内容都是公开的,所有不应该将隐私信息放到上面.免费.

gitpage是国外的服务器所以访问较慢.简单记录一下过程

1, 到GitHub上注册一个账号

注册时候username很关键,是你的账户名,后面要用它来做域名.注意name这个是昵称,两个不是一个东西.

2, 创建gitpage仓库

仓库:放东西的地方.

  • 点右上加号,选择new repository

  • 进入之后,可以看到Owner和’Repository name这一行. 在Repository name中输入'<username>.github.io这里的<username>就是上面提到的你的账户名,Repository name里填写的就是你的博客的访问网址.

  • 勾上Initialize this repository with a README

  • 点Create repository创建成功

3, 安装博客框架

这步做好博客就基本能用了.

11, 安装git和node.js

  • git:上传,下载,修改用的工具

  • node.js是支持博客的环境.

搜索安装就行了

22, 安装hexo框架

博客框架,装了就能搞博客,是不是很nb?还有其他类型的框架,喜欢哪个装哪个,装不出来概不负责

  • 找一个文件来放博客,以后你的博客的所有东西就在那个文件夹里, 在文件夹里右键选择Git Bash Here,,在黑框里输入npm install hexo-cli -g

然后依次输入命令然后回车(真傻瓜式操作..)

  • hexo init
  • npm install
  • hexo server这条命令让你在电脑上线下运行博客,在你更换修改主题后可以先用这个看看效果.

执行完后打开浏览器,网址栏输入http://localhost:4000/ , 就能看到你的博客的样子了.

在你的git黑框里按Ctrl C可以关掉本地模拟.

3, 更换主题

觉得hexo的版型不好看?东西有点少?到hexo官网下载喜欢的主题.

怎么下载?

先找一个喜欢的主题, 然后在他的博客里看他有没有放出github的链接,没有的话就只有去github上搜主题的名字,点进去之后长这样:

点右边那个绿色按钮Clone or Download,然后你会看到一个网址和一个下载zip的选项, 这是两种方式

  • 网址方式:在你的博客文件里打开git bash然后输入git 网址就行了
  • zip方式:把压缩包下载下来,然后解压到博客目录\themes\文件夹里面把解压出来的文件夹名字改个好记得名字,下面要用到它

怎么更换主题?

在博客目录下找到博客的配置文件_config.yml(可以用文本编辑器打开它,比如记事本或者notepad++.博客有这个配置文件,在主题文件夹里面也有_config.yml文件,但它们是各自的,不要混为一谈)

  • 在博客的_config.yml文件最后面找到theme:landscape把landscape改成上面你下载好放在themes文件夹里的那个主题的文件夹名字就行了,然后就可以先用上面提到的hexo server命令先看看效果了

博客的其他配置和主题的配置都一样,都是在各自的_config.yml文件里,后面慢慢配置成你喜欢的吧,想改哪里查哪里,因为真的不想什么都学啊!

怎么发表新文章?

在博客目录\source\_post\文件里就是文章,是Markdown格式的md文件. 用typora这个程序可以写,用别的也行,写完之后继续执行上面的三连就可以了

结语

这是一篇没什么技术含量的流水账.写于19年11月下旬.至于为什么有这篇流水账,因为有时候只需要结果而已,过程的背后细节对于无心在此深入的人没有大多意义. 对于一个没有it背景的人说那些框架如何如何有什么用?只是想有个自己的博客而已.

另外觉得gitpage太慢想要有自己的域名的,可以在github上重新绑定,具体就不在这里赘述了.
